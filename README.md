# Run and deploy the My Pinter Assistant app

This contains everything you need to run your app locally.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

### Run on web

1. go to https://mypinter.netlify.app/
